# 45-在线海鲜商城系统

[文档地址](http://wechat.zjrcsy.cn/)


##### 技术栈:springBoot+Vue+Mysql+MyBatis



##### 用户端: 

1.首页:轮播图展示、商品信息展示、秒杀商品展示、商城资讯展示

2.商品信息：展示商品列表，可按分类及名称、品牌、价格区间进行搜索查看，点击进入商品详情页可加入购物车或购买

3.秒杀商品：展示秒杀商品列表，可按分类及名称、品牌、价格区间进行搜索查看，点击进入商品详情页可加入购物车或购买

4.商城资讯：可查看资讯内容

5.购买车：可查看及下单

6.个人中心：查看我的订单我的地址和收藏。修改个人信息



**商家端：**

1.个人中心：个人信息修改及密码修改
2.商品信息管理：上架商品以及修改和删除

3.秒杀商品管理：上架秒杀商品以及修改和删除

4.订单管理：查看相关商品各个状态下的订单信息



##### 管理员端: 

1.个人中心：个人信息修改及密码修改

2.用户管理：对用户的增删改查

2.商家管理: 对商家的增删改查

3.商品分类管理:  对商品的分类进行操作

4.商品信息管理:  对商品的信息进行操作

5.秒杀商品管理:  对秒杀商品的信息进行操作

6.系统管理：商城资讯的发布、用户端轮播图的管理
